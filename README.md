# Adidas U.S. Sales Analysis 
## Project Objective
Using 2020-2021 data to record Adidas's metrics indicating which areas drive most revenue, divided by product category, retailer, etc. solely using Excel. 
Goal is to create business insights on what Adidas is excelling in and what they can improve on .

## Project Description 
Sales analysis is crucial for large corporations such as Adidas. Pre-processing, analyzing large datasets such as these can help us evalue KPIs (Key Performane Indicators), analyze important metrics, and create a dashboard that visualizes certain trends, sales performance, and profit margins.
